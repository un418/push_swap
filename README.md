# push_swap
Because Swap_push doesn’t feel as natural


Instructions:

How to use this program : 
The program only respond with bash terminal
if you use ZSH, to switch insert Bash in your terminal command.

Example: ARG="5 4 3 2 1"
./push_swap $ARG.

or simply ./push_swap 5 4 3 2 1

wrong use example; ./push_swap ARG="5 4 3 2 1"  or ./push_swap "5 4 3 2 1".

If those instruction are not respected, an ERROR will be print or expect Undifine Behavor.
